# v0.1.1

## What changed

- Packaged the default rule set with the Python package so installed CLI entrypoints can load rules without relying on a repo-relative `rules/default.json`.
- Improved auth-hint detection for common secret environment variable names such as `API_KEY`, `TOKEN`, and `SECRET_KEY`.
- Added clearer finding details for command, scope, and unauthenticated-network findings.
- Added regression tests for packaged rules and auth-hint behavior.

## Example

```bash
PYTHONPATH=src python3 -m mcp_tool_risk_audit examples/mcp-servers-map.json --format markdown --fail-on none
```
