# v0.1.0

## What changed

- Added first working Python CLI package for MCP risk audits.
- Added parser support for `mcpServers` map and list-style manifests.
- Added default risk rules in `rules/default.json`.
- Added markdown/JSON output, `--output` sink, and `--fail-on` behavior.
- Added unit tests and GitHub Actions CI.

## Example

```bash
PYTHONPATH=src python3 -m mcp_tool_risk_audit examples/mcp-servers-map.json --format markdown
```
